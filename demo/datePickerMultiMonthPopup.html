jquery.datePicker multi month plugin plugin example: showing how you can use the datePickerMultiMonth plugin as a popup

This page illustrates the use of the "plugin plugin" datePickerMultiMonth. It is a plugin which builds on the functionality of the date picker to allow you to have datePickers which display multiple months at a time. The construction of the multimonth plugin illustrates the flexiblility of the datePicker and how you can add functionality without increasing the filesize of the base plugin.
